# Password Policy Evaluator 

A lightweight, client-side web application designed to assess organizational password policies against modern cybersecurity standards (such as NIST SP 800-63B guidelines). 

This tool provides IT administrators and security professionals with a quick security score, a risk assessment (Low, Medium, High), and actionable recommendations to improve their credential requirements.

## Features
* **Instant Assessment:** Calculates a security score (0-100) based on modern password standards.
* **Actionable Recommendations:** Generates specific, rule-based feedback (e.g., advising against frequent password rotation, emphasizing MFA and longer passphrases).
* **Privacy-First (No Backend):** 100% client-side JavaScript. No data is transmitted to any server, ensuring sensitive organizational policy details remain entirely private.
* **Responsive Design:** Clean, modern UI built with CSS Grid that works across desktop and mobile browsers.

##  How to Use
Because this is a static web application, no server or installation is required.
